# 图标文件说明

此目录需要包含以下图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

## 获取图标

### 方法1：使用在线工具生成

1. 访问 https://www.favicon-generator.org/
2. 上传图片或输入文字
3. 下载生成的图标包
4. 提取 PNG 文件到此目录

### 方法2：使用图片编辑工具

使用 Photoshop、GIMP 等工具创建：
- 16x16 像素的 PNG 图片
- 48x48 像素的 PNG 图片
- 128x128 像素的 PNG 图片

### 方法3：使用占位图片

如果没有图标，可以创建简单的纯色图片作为占位符。

## 图标设计建议

- 使用简洁的设计
- 确保在小尺寸下也能清晰识别
- 建议使用与 Dify 品牌色相近的颜色
- 可以使用备份/下载相关的图标元素
